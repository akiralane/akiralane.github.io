Enums

Sometimes you want a variant type just for its labels - consider the common example of days of the week.

In this case, having to use values in a variant type and giving a type to each of the labels would be annoyingly verbose. Instead, we use the labels and the unit type:

data Day = Mon :: () | Tue :: () | ... | Sun :: ()

An enumerated type is simply sugar for exactly this structure.

This also means you can mix and match regular variant types and enums - consider Maybe:
