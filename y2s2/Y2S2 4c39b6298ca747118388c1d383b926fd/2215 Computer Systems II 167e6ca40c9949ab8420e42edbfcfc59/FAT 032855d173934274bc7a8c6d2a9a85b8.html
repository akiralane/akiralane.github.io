What does the actual FAT look like?

The file system is obviously an important part of the operating system. It’s one out of the three letters in DOS!

It handles file structure, but it’s also responsible for naming, access and operations on files.

The block abstraction

The big draw of the block abstraction is that the memory can be assembled from non-consecutive portions, whereas with raw memory you’d have to read from a consecutive part (and potentially run out of space thanks to fragmentation).

FAT

FAT (named after the File Allocation Table it uses as part of its metadata).

It’s nice because it doesn’t use much RAM (since you only need to copy the FAT into memory). RAM was expensive, and they wanted to squeeze as much out as possible.

The implementation is also pretty simple.

It’s not as scalable as modern filesystems, but since it’s so straightforward it’s supported by pretty much everything from its inception to now.

Good as a cross-platform storage medium!

FAT system layout

Master Boot Record (MBR)

A master boot record is a type of boot sector at the very beginning of partitioned storage devices. It holds the metadata on how the logical partitions are physically organised on the medium.

Partition

Directory entry

The name is 11 bytes for a short “8:3” filename : nnnnnnnn.txt.

To delete a file, the first byte of the name in the directory is changed to 0xe5 and the blocks are marked as “unused” in the FAT.

This means that, as long as nothing has been written to the system afterwards, all the data is still actually present!

Further, if the medium was fresh (i.e. no data was there originally), then the blocks will be consecutive. This means that (since you can recover the filesize from the directory), 100% of the data is recoverable.

What does the actual FAT look like?

The first entry is reserved for media type, the second is reserved for partition state.

The partition state:

FFFF → clean (set on shutdown)

FFF8 → dirty (set on write and rename)

If it’s dirty on startup, then it’s possibly corrputed.
