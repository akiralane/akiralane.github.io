What if we wanted to compute the probability it rained, given the evidence that the grass is wet?

We know that P(R,S,W)=P(R)P(S)P(W∣S,R)P(R,S,W)=P(R)P(S)P(W|S,R)P(R,S,W)=P(R)P(S)P(W∣S,R)﻿, although I don’t think that’s relevant here.

This comes down to simply using Bayes’ Theorem:

P(A∣B)=P(B∣A)P(A)P(B)P(A|B)={P(B|A)P(A)\over{P(B)}}P(A∣B)=P(B)P(B∣A)P(A)​﻿

which we can substitute as:

P(r∣w)=P(w∣r)P(r)P(w)P(r|w)={P(w|r)P(r)\over{P(w)}}P(r∣w)=P(w)P(w∣r)P(r)​﻿

If we sum over all the possible probabilities that W=wW=wW=w﻿ dependent on the other two, we can extract P(w)P(w)P(w)﻿:

P(r∣w)=P(w∣r)P(r)Σr′s′P(w∣r′s′)P(r′)P(s′)P(r|w)={P(w|r)P(r)\over{\Sigma_{r's'}P(w|r's')P(r')P(s')}}P(r∣w)=Σr′s′​P(w∣r′s′)P(r′)P(s′)P(w∣r)P(r)​﻿

Consider that the probability that the grass is wet given that it’s rained or the sprinkler has been on is always 111﻿. This means that we can just kinda ignore those factors.

The only time this isn’t the case is when R=¬rR= \neg rR=¬r﻿ and S=¬sS= \neg sS=¬s﻿, at which point it’s 000﻿. This makes the P(¬r)P(¬s)P(\neg r)P(\neg s)P(¬r)P(¬s)﻿ term disappear.

P(r∣w)=P(r)P(r)P(s)+P(¬r)P(s)+P(r)P(¬s)P(r|w)={P(r)\over{P(r)P(s)+P(\neg r)P(s)+P(r)P(\neg s)}}P(r∣w)=P(r)P(s)+P(¬r)P(s)+P(r)P(¬s)P(r)​﻿

Notice how this is just P(r)P(r)P(r)﻿ on the numerator. This means that the probability that it rained given that it was wet is greater than or equal to the probability that it rained. P(r∣w)≥P(r)P(r|w)\ge P(r)P(r∣w)≥P(r)﻿.

Multiple causes - one among many is observed

What if we wanted to find the probability that it rained, but we knew both that the grass was wet AND the sprinkler was on?

We just have to make everything additionally dependent on sss﻿, since we have the information that S=sS=sS=s﻿:

P(r∣w,s)=P(w∣r,s)P(r∣s)P(w∣s)P(r|w,s)={P(w|r,s)P(r|s)\over{P(w|s)}}P(r∣w,s)=P(w∣s)P(w∣r,s)P(r∣s)​﻿

But we said earlier that P(w∣r,s)P(w|r,s)P(w∣r,s)﻿ and P(w∣s)P(w|s)P(w∣s)﻿ are just 111﻿. Further, since we know that the sprinkler was on, P(r∣s)=P(r)P(r|s)=P(r)P(r∣s)=P(r)﻿.

This gets us:

P(r∣w,s)=1×P(r)1=P(r)P(r|w,s)={1\times P(r)\over{1}}=P(r)P(r∣w,s)=11×P(r)​=P(r)﻿

So, the probability it rained given that it was wet AND the sprinkler was on is simply the probability that it rained.

That is, if we know that the sprinkler was on, we can’t make any assumptions about the rain because we know that the grass is wet due to the sprinkler. Hence, the rain’s probability is just the probability that it rained on any normal day.
