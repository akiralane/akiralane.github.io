Interlude: Demon Game

The Demon Game is a structure for a proof that a language is not regular, based on the idea that you can frame the pumping lemma as an adversarial game.

Example

is not regular.

Note that we can see this intuitively: the size of the string is unbounded and in order to uphold the condition, we have to 'remember' at least one of mmm﻿ or nnn﻿ while traversing the string. You can't do this with a finite number of states, so there is no corresponding DFA and the language cannot be regular.

Watch out though - remember that there can still be regular langauges which are described by unbounded numbers. DFA can have loops.
