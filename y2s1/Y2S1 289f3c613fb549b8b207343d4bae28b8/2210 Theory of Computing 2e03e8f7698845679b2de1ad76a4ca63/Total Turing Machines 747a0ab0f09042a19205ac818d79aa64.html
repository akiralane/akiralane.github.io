Total Turing Machines

A TM is said to halt on an input if it either accepts or rejects that input.

It's possible that the machine neither accepts nor rejects - it loops.

A TM is total if it halts for all inputs.

As you can intuit from the relation to TMs, every recursive set is recursively enumerable, but not vice versa.
