All regular languages are context free, but not all languages are context free, obviously.

For example, {anbncn∣n≥0}\{a^nb^nc^n| n\geq0\}{anbncn∣n≥0}﻿ is not context free.

We're going to show how to prove this, through the CFL pumping lemma.

The pumping lemma for CFLs

For every context-free language AAA﻿, there exists a k≥0k\geq0k≥0﻿ such that every string z∈Az\in Az∈A﻿ of size at least kkk﻿ can be broken up into five substrings:

z=uvwxyz=uvwxyz=uvwxy﻿

such that vx≠εvx\neq \varepsilonvx=ε﻿ and #(vwx)≤k\#(vwx)\leq k#(vwx)≤k﻿.

Also, for all i≤0,i\leq 0,i≤0,﻿

uviwxiy∈Auv^iwx^iy\in Auviwxiy∈A﻿

We'll prove this by examining derivations of sentences in CFGs which are in Chomsky Normal Form.

Parse trees

A parse tree for a string zzz﻿ is a tree representing the productions applied in a derivation from SSS﻿, independent of the order of application.

Here's an example of a parse tree for a4b4a^4b^4a4b4﻿ in the grammar {anbn∣n≥1}\{a^nb^n|n\geq1\}{anbn∣n≥1}﻿:

a→AC∣ABa\rarr AC|ABa→AC∣AB﻿

C→SBC\rarr SBC→SB﻿

A→aA\rarr aA→a﻿

B→bB \rarr bB→b﻿

Let's extend the parse trees so that the terminals keep producing themselves down to the the bottom level. This doesn't change the tree semantically, so it's ok:

Notice that, at each level (for grammars in CNF), we have at most twice as many symbols as the previous level.

For the bottom level to have at least 2n2^n2n﻿ symbols, the tree must have depth of at least nnn﻿ (aka at least n+1n+1n+1﻿ levels).

The depth of the tree is defined as the number of edges on the longest path from the root to any leaf. Note that the depth is going to be one greater than the level of the lowest leaf, since the levels are zero-indexed.

Proof using parse trees

Let GGG﻿ be a grammar in CNF.

Let k=2n+1k=2^{n+1}k=2n+1﻿, where nnn﻿ is the number of non-terminals.

Suppose there is some z∈Az\in Az∈A﻿ and that #(z)≥k\#(z) \geq k#(z)≥k﻿

Any parse tree for zzz﻿ must have depth of at least n+1n+1n+1﻿. However, the path also has length at least n+1n+1n+1﻿, so some non-terminals must occur more than once.

Hence, let XXX﻿ be a non-terminal with two occurences.

We can break zz z﻿ into uvwxyuvwxyuvwxy﻿ where www﻿ is generated by the lower occurence of XXX﻿ and vwxvwxvwx﻿ is generated by the upper occurence of XXX﻿.

Then:

Let TTT﻿ be the subtree rooted in the upper occurence of XXX﻿.

Let ttt﻿ be the subtree rooted in the lower occurence of XXX﻿.

We can replace tt t﻿ with TTT﻿ to get a parse tree for uv2wx2yuv^2wx^2yuv2wx2y﻿.

Since TTT﻿ contains ttt﻿, we can repeat this process as many times as we like to get a parse tree for uviwxiyuv^iwx^iyuviwxiy﻿ for any i≥1i\geq 1i≥1﻿.

Notice that #(vwx)≤k\#(vwx)\le k#(vwx)≤k﻿.

This proves the pumping lemma for CFLs - all the conditions in the lemma are met.

Contrapositive and demon game

Like the regular language pumping lemma, we can form the contrapositive to prove that a language is not context-free. Also like the regular languages one, we can understand it as a Demon Game:

  1. The demon picks k≥0k\ge 0k≥0﻿.
  1. You pick z∈Az\in Az∈A﻿ of length at least kkk﻿.
  1. The demon breaks zzz﻿ into uvwxyuvwxyuvwxy﻿ such that:
    1. #(vx)>0\#(vx)\gt0#(vx)>0﻿
    1. #(vwx)≤k\#(vwx)\le k#(vwx)≤k﻿
  1. You pick i≥0i\geq 0i≥0﻿, winning if uviwxiyuv^iwx^iyuviwxiy﻿ is not in the language AAA﻿.

In the Demon Game for CFLs, you have to be more concerned about each specific case, rather than proving it generally.

For this, it's useful to think of vwxvwxvwx﻿ as a "window" of length ≤k\le k≤k﻿ which slides along your string (as demonstrated by the image in the example underneath).

Remember that each of vvv﻿, www﻿ and xxx﻿ are strings in their own right, not characters. They can contain a combination of different characters from the alphabet.

Example 1

Consider {anbncn∣n≥0}\{a^nb^nc^n|n\ge 0\}{anbncn∣n≥0}﻿.

  1. The demon picks a k≥0.k\geq0.k≥0.﻿
  1. I pick z=akbkck.z=a^kb^kc^k.z=akbkck.﻿ This has a length of at least kkk﻿ and is in the language AAA﻿.
  1. The demon breaks zzz﻿ into uvwxyuvwxyuvwxy﻿ where vxvxvx﻿ is not empty and the length of vwxvwxvwx﻿ is greater than kkk﻿.
  1. I pick i=2i=2i=2﻿. We now have to consider the possible choices the demon could have made in step 3, when zzz﻿ is uv2wx2yuv^2wx^2yuv2wx2y﻿. This is a bit of a pain compared to the process regular languages.

In the above cases, which are exhaustive, we have escaped the language. Hence, we have proven that the language is not context-free via contradiction.

Example 2

Prove that {anbmanbm∣n,m≥0}\{a^nb^ma^nb^m|n,m\ge0\}{anbmanbm∣n,m≥0}﻿ is not context-free.

Time for a demon game:

  • Demon takes some k≥0k\ge0k≥0﻿.
  • We will pick z=akbkakbkz=a^kb^ka^kb^kz=akbkakbk﻿. This is in the language; we chose the same value for nn n﻿ and mmm﻿.
  • The demon breaks zzz﻿ into uvwxyuvwxyuvwxy﻿ following the conditions:
    • #(vx)>0\#(vx)\gt0#(vx)>0﻿
    • #(vwx)≤k\#(vwx)\leq k#(vwx)≤k﻿
  • We now have to pick an iii﻿ for uviwxiyuv^iwx^iyuviwxiy﻿ in order to escape the language or break the conditions. Note that we can pick an iii﻿ for each condition - it doesn't have to be the same iii﻿!
    • i=2i=2i=2﻿ seems to be the panacea for CFL demon games, so we're going to go with that again.
    • Remember to think of vwxvwxvwx﻿ as a string "window" of length at most kkk﻿.

Here are the demon's options:

  • Either vvv﻿ or xxx﻿ contains both aaa﻿'s and bbb﻿'s.
    • i=2i=2i=2﻿; doubling any sort of pattern which contains both aaa﻿'s and bbb﻿'s would break the form of the language a∗b∗a∗b∗a^*b^*a^*b^*a∗b∗a∗b∗﻿. Hence, we win.
  • vvv﻿ and xxx﻿ contain only aaa﻿'s.
    • When we pump vvv﻿ and xxx﻿, we break the symmetry of the language - the first set of aaa﻿'s has a different number of aaa﻿'s to the second. Hence, we win.
  • vvv﻿ and xx x﻿ contain only bbb﻿'s has an identical argument.
  • vvv﻿ contains only aaa﻿'s and xxx﻿ contains only bbb﻿'s.
    • This is similar to the previous case - when we pump vvv﻿ and xxx﻿, we are breaking the symmetry of the language, because one side of the language will have different sized groups of aaa﻿'s followed by bbb﻿'s to the other.
  • vvv﻿ contains only bbb﻿'s and xxx﻿ contains only aaa﻿'s has an almost identical argument. Think about where this vwxvwxvwx﻿ would end up (like in the pictures so far). There's only one place it can go.

This is every option. We win every time - the language is not context free.
