Remember Backus-Naur form? It's a way of defining context free grammars.

A context free grammar (CFG) is a finite set of rules which define a well-formed expression. This produces a language of expressions.

Here's an example of a Backus-Naur Form CFG for a simple programming language:

Terminals and non-terminals

Terminals are expressions which cannot be replaced further - things like 0, 1, +, <, a in the above CFG.

Non-terminals are expressions which have 'holes' in - as in, they can be expanded further. In the above CFG, things like if <bool-expr> then <stmt> else <stmt> and <arith-expr> are non-terminals. As their name suggests, non-terminals don't exist in the final sentence - they all get replaced with appropriate terminals.

Derivation

Derivation is the process of creating a well-formed sentence using the rules of the CFG, replacing non-terminals. You typically begin with a generic starting non-terminal, usually denoted SSS﻿.

In this example using the above BNF CFG, <stmt> is the starting symbol.

Formal definition

A context-free grammar GGG﻿ is a quadruple (N,Σ,P,S)(N,\Sigma,P,S)(N,Σ,P,S)﻿:

  • NNN﻿ is a finite set of non-terminals.
  • Σ\SigmaΣ﻿ is a finite set of terminals.
  • PPP﻿ are the productions - also called rules.
    • The type of the set of productions is: P⊆N×(N∪Σ)∗P\sube N\times(N\cup\Sigma)^*P⊆N×(N∪Σ)∗﻿. That is, pairs of (A,α)(A,\alpha)(A,α)﻿ telling us which non-terminals you can replace which terminals with.
    • Remember that α\alphaα﻿ is a string. See the symbols heading below.
  • SSS﻿ is the starting non-terminal.
    • S∈NS\in NS∈N﻿. Notice that this isn't a set, despite being a capital letter! This is due to how we notate terminals and non-terminals.

Symbols

Non-terminals: capital letters - A,B,C,…A,B,C,\dotsA,B,C,…﻿

Terminals: lowercase letters - a,b,c,…a,b,c,\dotsa,b,c,…﻿

Strings: lowercase greek letters - α,β,γ,…\alpha,\beta,\gamma,\dotsα,β,γ,…﻿

Note that strings are in the set (N∪Σ)∗(N\cup\Sigma)^*(N∪Σ)∗﻿ - that is, strings may contain non-terminals.

Context-free languages

The language of a context free grammar is given by L(G)={x∈Σ∗∣S⇒x}L(G)=\{x\in\Sigma^*|S\Rarr x\}L(G)={x∈Σ∗∣S⇒x}﻿.

A language is context-free if there exists a context-free grammar which can produce it.
