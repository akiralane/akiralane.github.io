Turing Machines

Informally, a Turing machine MM M﻿ has a tape and a head. The tape has a semi-infinite number of cells on it (bounded on the left, hence 'semi' infinite) and the head moves left and right on the tape, reading and writing symbols.

The input string is of finite length, written on contiguous cells. The remainder of the cells on the tape are filled with the blank symbol ⊔\sqcup⊔﻿. A cell before the input is written holds the start symbol ⊢\vdash⊢﻿.

Internally, the machine has states - the transitions of which are determined by what the head reads from the tape. At each step, MMM﻿ reads the symbol on the tape under the head, writes a new symbol and moves either left or right.

The symbol which gets written and the movement taken is determined by a transition function δ\deltaδ﻿.

MMM﻿ accepts as soon as it enters accept state ttt﻿.

MMM﻿ rejects as soon as it enters reject state rrr﻿.

This means that it can loop indefinitely without rejecting or accepting, depending on the machine and input.

Formal Notation

A deterministic one-tape TM is a 9-tuple:

M=(Q,Σ,Γ,⊢,⊔,δ,s,t,r)M=(Q,\Sigma,\Gamma,\vdash,\sqcup,\delta,s,t,r)M=(Q,Σ,Γ,⊢,⊔,δ,s,t,r)﻿

where:

  • QQQ﻿ is the finite set of states.
  • Σ\SigmaΣ﻿ is the finite set of symbols in the input alphabet.
  • Γ\GammaΓ﻿ is the finite set of symbols in the tape alphabet, such that Σ⊆Γ\Sigma\sube\GammaΣ⊆Γ﻿.
  • ⊢\vdash⊢﻿ is the left endmarker, an element of Γ∖Σ\Gamma\setminus\SigmaΓ∖Σ﻿.
  • ⊔\sqcup⊔﻿ is the blank symbols, an element of Γ∖Σ\Gamma\setminus\SigmaΓ∖Σ﻿.
  • δ\deltaδ﻿ is the transition function such that δ:Q×Γ→Q×Γ×{L,R}\delta:Q\times\Gamma\to Q\times\Gamma\times\{L,R\}δ:Q×Γ→Q×Γ×{L,R}﻿
  • sss﻿, ttt﻿ and rrr﻿ (where r≠tr\ne tr=t﻿) are the start, accept and reject states respectively. They are all elements of QQQ﻿.

Transitions

A transition is determined by the transition function:

δ(p,a)=(q,b,d)\delta(p,a) = (q,b,d)δ(p,a)=(q,b,d)﻿

The above means:

"when in state ppp﻿ reading aaa﻿ off the tape, replace it with bbb﻿, move in direction ddd﻿ and enter state qqq﻿"

The TM will never move off the tape to the left of the endmarker, nor will it ever leave an accept or reject state.
