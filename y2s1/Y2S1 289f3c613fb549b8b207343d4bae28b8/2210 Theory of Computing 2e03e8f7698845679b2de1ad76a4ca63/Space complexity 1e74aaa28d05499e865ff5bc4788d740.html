Recall that the time complexity of a decider MMM﻿ is the function fff﻿ where f(n)f(n)f(n)﻿ is the maximum number of steps MMM﻿ uses on any branch of computation on any input of length nnn﻿.

Space complexity is pretty similar.

The space complexity of a decider MMM﻿ is the function fff﻿ where f(n)f(n)f(n)﻿ is the maximum number of tape cells MMM﻿ scans on any branch of its computation, on any input of length nnn﻿.

We say that MM M﻿ is an f(n)f(n)f(n)﻿ space Turing machine.

The classes SPACE and NSPACE

The complexity class SPACE(f(n)f(n)f(n)﻿) is the collection of all languages which are decidable by an O(f(n))O(f(n))O(f(n))﻿ space deterministic Turing machine.
The complexity class NSPACE(f(n)f(n)f(n)﻿) is the same, but the Turing machine is non-deterministic. Who’d have seen that one coming?

Note that this isn’t one single class - the class depends on the function: NSPACE(n2n^2n2﻿) is not the same class as NSPACE(n!n!n!﻿), even though they both have NSPACE in big capitals. The function is a significant component.

Relating space and time complexity

If a decider runs in f(n)f(n)f(n)﻿ space, then it runs in 2O(f(n))2^{O(f(n))}2O(f(n))﻿ time.

Look at “The cost of determinism” in the linked section for a refresher on the “two to the big O” thing.

PSPACE and NPSPACE

Oh god, we’re going even deeper.

Note that these don’t relate to a function, unlike SPACE and NSPACE, which do. Don’t make a typo!

PSPACE is the set of all languages which can be decided by a deterministic Turing machine using polynomial space.

NPSPACE is the same, but with a non-deterministic Turing machine.

Savitch’s theorem: PSPACE = NPSPACE

Things are different around here.

Savitch’s Theorem: Every f(n)f(n)f(n)﻿ space non-deterministic Turing machine has an equivalent f(n)2f(n)^2f(n)2﻿ space deterministic Turing machine.

Relationships between complexity classes

P ⊆\sube⊆﻿ PSPACE: a deterministic TM which runs in polynomial time can only use polynomial space.

NP ⊆\sube⊆﻿ NPSPACE: similar reason as above.

Using Savitch’s Theorem (PSPACE and NPSPACE are equivalent), NP ⊆\sube⊆﻿ PSPACE.

PSPACE ⊆\sube ⊆﻿ EXPTIME: If a decider runs in f(n)f(n)f(n)﻿ space, it runs in 2O(f(n))2^{O(f(n))}2O(f(n))﻿ time. Again, look above for a refresher on what this means.

We don’t actually know which of these inclusions are strict!

We do know that P ⊂\sub⊂﻿ EXPTIME, though, since we can define things which are explicitly exponential and these are provably irreducible to polynomial.

You can also define EXPSPACE, if you like pain.
