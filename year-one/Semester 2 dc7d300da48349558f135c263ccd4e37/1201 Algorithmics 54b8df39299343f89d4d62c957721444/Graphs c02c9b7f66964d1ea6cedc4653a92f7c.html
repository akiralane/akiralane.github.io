Bipartite graph: the vertices can be partitioned into two "classes" such that every edge from one class ends in the other. You can sort of think of this as representing a relational image between a domain and co-domain.

Simple graph: finite, unweighted, undirected, acyclic, single edge between vertices

Complete graph: a simple graph where every pair of vertices are connected by an edge.

k-regular graph: every vertex of the graph has the same degree kkk﻿. The degree is the number of edges from a vertex.

Planar graph: can be drawn with no edges crossing

Directed graph: the edges have direction

Weighted graph: the edges have numerical values associated with them, referred to as weights.

Moving through graphs

Walk: a sequence of alternating vertices and edges of a graph - i.e. moving through the graph

Trail: if the edges of a walk are distinct, then the walk is a trail.

Path: if the vertices of a trail are distinct, then it is called a path.

A circuit is a closed trail, while a cycle is a closed path.

Adjacency Matrix

You can represent a graph as an N×NN\times NN×N﻿ matrix where NNN﻿ is the number of nodes in the graph. The value of the matrix element i,ji,ji,j﻿ is 111﻿ if iii﻿ has an edge to jjj﻿.

In an undirected graph, this matrix will be symmetrical because the property of having an edge is symmetric in that case.

Eulerian graphs

An Euler trail travels every edge exactly once. If it ends at the initial vertex, then it's an Euler circuit.

A graph is Eulerian if an Euler circuit exists in it.

How do we know if a graph is Eulerian?

You'd think that this would take some search algorithm - however, this can actually be done analytically.

We observe that, in any Eulerian graph, all (or almost all) of the vertices have an even degree.

In fact,

If GGG﻿ has no odd-degree vertices, then it admits an Euler circuit.

If GGG﻿ has only two odd-degree vertices, then it admits an Euler trail.

Ends up being pretty easy!

Hamiltonian graphs

A Hamiltonian path is a path which visits every vertex exactly once. If it ends at the initial vertex, then it's a Hamiltonian cycle.

A graph is Hamiltonian if it admits a Hamiltonian cycle.

Unfortunately, there's no simple characterisation of the existence of a Hamiltonian path/cycle. In fact, determining this is an NP-Complete problem!
