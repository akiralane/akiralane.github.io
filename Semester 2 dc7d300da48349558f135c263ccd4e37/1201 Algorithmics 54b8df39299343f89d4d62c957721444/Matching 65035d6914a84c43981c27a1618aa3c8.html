Matching

A matching is a set of edges without any common vertices. That is, it's an independent edge set.

A set of vertices is "covered" by a matching MM MÔªø is every vertex in the set is incident with an edge in MMMÔªø.

A maxiMUM (cardinality) matching is a matching that contains the largest possible number of edges.

Bipartite Graphs

Recall that a graph is bipartite iff it is 2-colourable. (You can find a bipartite graph visually if you can colour it with two colours, where no two adjacent nodes have the same colour). That is, it's bipartite iff it contains no odd cycles.

That means our maximum matching algorithm will find a maximum matching.

Blossoms

Given a non-bipartite graph GGGÔªø and a matching MMMÔªø, a blossom is an odd cycle where one of its vertices is such that there exists an alternating path of even length from the vertex to an unmatched vertex. The alternating path is named the stem.

Algorithm Design thoughts

  • Brute force
    • Usually prohibitively expensive.

    • Short-sighted; sometimes local optimum is global optimum, but sometimes it isn't.
  • Optimal algorithms
    • Need insights and rigor proofs
  • Approximation algorithms
