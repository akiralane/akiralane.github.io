Keys

A set of attributes forms a key for a relation if we do not allow two different tuples in a relation instance to have the same values in all attributes of the key.

So, a key is a set of attributes which have a unique combination for each "entry". Consider the following relation:

Movies(title, year, length, genre, studioName, starName)
