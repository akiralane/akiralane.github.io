NFA to regexp

We want some regex α\alphaα﻿ which describes all strings in an NFA MMM﻿.

Turns out the set of regular expressions is defined inductively.

Base cases:

  • Every σ∈Σ\sigma \in \Sigmaσ∈Σ﻿ is a regular expression.
  • ε\varepsilonε﻿ is a regular expression.
    • The language of the empty string is the singleton set of the empty string
  • ∅\empty∅﻿ is a regular expression.
    • The language of the empty set is the empty set

Operations:

  • If α\alphaα﻿, β\betaβ﻿ are regexps, then α+β\alpha + \betaα+β﻿ is also a regexp.
    • The plus is 'or'. This is often written '∣|∣﻿'.
    • L(α+β)=L(α)∪L(β)L(\alpha + \beta) = L(\alpha) \cup L(\beta)L(α+β)=L(α)∪L(β)﻿
