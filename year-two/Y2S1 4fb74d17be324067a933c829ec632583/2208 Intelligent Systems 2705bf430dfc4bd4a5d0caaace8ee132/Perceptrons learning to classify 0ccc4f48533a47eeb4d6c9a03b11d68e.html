How does this fit into the general story?

  • The perceptron algorithm is based on updates driven by misclassification.
  • If there aren't any errors left, then there are no more iterations. One it's done with the training data, it's done.
  • We only converge on a solution in a perceptron if the data is linearly separable. It's hard to tell if the perceptron will converge for any given set of data.
  • Even if the data is linearly separable, there could be multiple solutions for wwwÔªø.
